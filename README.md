# bus_reservation_system.c
A console-based Bus Reservation System developed in C using Data Structures. It allows users to reserve seats, view bus details, and manage bookings through a menu-driven interface.

# 🚌 Bus Reservation System
A console-based Bus Reservation System developed in C using Data Structures.

## 📌 Features
- Bus seat reservation
- View bus details
- Display seat availability
- Passenger booking management
- Menu-driven interface

## 🛠 Technologies Used
- C Programming
- Data Structures
- GCC Compiler
